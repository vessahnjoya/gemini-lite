# BCS1120-GeminiProject

## Running project

     Step 1: Clone repository and checkout to resulting folder
     git clone https://gitlab.maastrichtuniversity.nl/I6371413/bcs1120-geminiproject.git
     cd BCS1120-GEMINIPROJECT

     Step2: Run the following command to create the jar file
     mvn clean package

     Step3: ruRunn client as follows 
     java -cp target/bcs2110-2025.jar gemini_lite.Client <URL> [<input>]
